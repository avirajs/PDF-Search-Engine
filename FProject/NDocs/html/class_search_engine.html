SearchEngine::SearchEngine ( )

The SearchEngine class uses the various class to create a PDF Search Engine. The class makes use of the DocumentParser, IndexHandler, and QueryEngie class to perform all of the various tasks of the Mantience mode and Query mode of the user interface.
